<h1>GroupIt</h1>

GroupIt is an group chatting app that requires no login or setup. 

<h1>Current features</h1>
* Creating groups
* Joining groups
* Separate chatting
* Picture attachments
* Does not require a lot of resources. (Does not lag)
* Nice design


<h1>Planned features</h1>
* Voice chatting
* File transfer
* Screensharing
* Encrypt message
